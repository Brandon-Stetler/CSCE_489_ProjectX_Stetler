Put ~20 photos in this folder. File names must start with a class key:
  orange_square_*.jpg
  blue_square_*.jpg
  orange_circle_*.jpg
  blue_circle_*.jpg

Examples:
  orange_square_01.jpg
  blue_square_03.jpg
  orange_circle_desk1.jpg

The script reads the first two underscore-separated tokens to decide the label.
